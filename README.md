# Iris Flower Classification

## Introduction
This project focuses on classifying Iris flowers into different species using Machine Learning. The classification is based on flower measurements such as sepal length, sepal width, petal length, and petal width.

## Dataset
Dataset Used: Iris.csv

### Features
- SepalLengthCm
- SepalWidthCm
- PetalLengthCm
- PetalWidthCm

### Target Variable
- Species
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

## Tools and Libraries Used
- Python
- Pandas
- Scikit-learn

## Objectives
- Use Iris flower measurements as input data.
- Train a Machine Learning model for classification.
- Classify flower species based on measurements.
- Evaluate model accuracy and performance.
- Understand basic classification concepts.

## Project Workflow
1. Data Loading
2. Data Exploration
3. Missing Value Check
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Species Prediction
8. Model Evaluation
9. Accuracy Calculation

## Results
The Random Forest Classifier was trained using flower measurements and successfully classified Iris flowers into their respective species with high accuracy.

## Conclusion
The Iris Flower Classification project demonstrates the use of Machine Learning for classification tasks. The model accurately predicts flower species based on sepal and petal measurements and helps understand fundamental classification concepts.
